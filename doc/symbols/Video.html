ビデオファイルを読み込む

Parameters:

							
								

									options.id
									
								

								
videoタグに一意につけられるID

							
								

									options.src
									
								

								
ファイルのURLを配列で指定(マルチブラウザ対応、ブラウザ内では指定したどれかを再生)

							
								

									options.width
									Optional
								

								
ビデオの横幅
ピクセル単位
省略時は640

							
								

									options.height
									Optional
								

								
ビデオの高さ
ピクセル単位
省略時は480

							
								

									options.left
									Optional
								

								
ブラウザ(もしくは親ブロックからの右方向の位置
単位はピクセル
省略時は0(ブロックの左端)

							
								

									options.top
									Optional
								

								
ブラウザ(もしくは親ブロックからの上方向の位置
単位はピクセル
省略時は0(ブロックの上端)

							
								

									options.z
									Optional
								

								
ビデオの奥行き
CSSのz-indexと同じ
省略時は0

							
								

									options.body
									Optional
								

								
ビデオを埋め込むブロック
jQueryオブジェクトを指定
省略時はbodyブロック($("body"))

							
								

									options.loop
									Optional
								

								
ビデオを繰り返し再生するかを指定
省略時はfalse(繰り返さない)

							
								

									options.autofinish
									Optional
								

								
再生終了時、自動的にvideoタグを取り除くかを指定
省略時はfalse(取り除かない)
