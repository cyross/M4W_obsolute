<static> AssetsLoader.load()
画像などのアセットを読み込み、指定した関数をコールバックする
すべてのロードに成功したときはsuccess関数、一つでも失敗したときはfailed関数を呼ぶ success関数の引数は以下のキーを持つ辞書オブジェクト
KeyValue
imageImageオブジェクトの辞書オブジェクト
spriteSpriteオブジェクトの辞書オブジェクト
bgmBGMのSoundオブジェクトの辞書オブジェクト
se効果音のSoundオブジェクトの辞書オブジェクト
failed関数の引数はdefer.when.failedメソッドの引数の配列が渡される
実際のImage/Sprite/Soundを取得するときは以下の方法でアクセス可能

例1)

function success(assets){

   var image = assets.image.(id);

   var sprite = assets.sprite.(id);

   var bgm = assets.bgm.(id);

   var se = assets.se.(id);

}

例2)

function success(assets){

   var image = assets["image"]["id"];

   var sprite = assets["sprite"]["id"];

   var bgm = assets["bgm"]["id"];

   var se = assets["se"]["id"];

}
Parameters:
options.assets
アセット設定オブジェクトの配列
options.assets[].type
アセットの形式
画像は"image"、スプライトは"sprite"、音声は"sound"で示される
options.assets[].その他
Image,Sprite,Soundクラスコンストラクタの引数と同じものを指定
options.success Optional
ロードが全て完了したときに呼び出されるコールバック関数
引数はアセット管理オブジェクト
内容は後述
options.failed Optional
ロードに失敗した時に呼び出されるコールバック関数
引数は失敗した時に渡される引数リスト
